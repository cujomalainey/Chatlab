Start
	LaunchClient - Launches The Client

Utility
	AddPath      - Add a subpath to the search directory
	ClearPath	 - Removes every path elements that is linked to a toolbox
	NewWindow    - Create a new figure window that is initialized

Network
	connect(host, port, receiveCallback, disconnectCallback)	- Creates a channel and connects to the server
	bindServer(host, port, receiveCallback, acceptCallback)		- Binds a channel to a port. Accepts all incoming connections
	disconnect(channel)											- Disconnects the channel from the other end and performs the cleanup
	sendMessage(channel, string)								- Send the string across the channel

Packets
	handshake(key, step)				- Create a handshake packet. step = Init, Response, Complete --> Client sends response
	loginRequest(username, password)	- Attempt to login
	loginResponse(success)				- The response from the server. success = True, False
